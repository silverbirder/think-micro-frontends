> Luigi is an open source JavaScript framework for micro frontends.

Luigiは、LuigiCoreを中心としたMicro Frontendsに必要な機能を提供するフレームワーク。
RoutingやAuthなどの機能や、Vue,React,Angularなどをサポートしているみたいだ。
